## 项目简介

LBTI 是一个面向所有人的趣味恋爱人格测试网站，灵感来自 MBTI 和 [CBTI](https://github.com/liyupi/cbti-test)。

通过 30 道和恋爱日常相关的选择题，从 **5 大行为模型、15 个维度** 综合分析，匹配出你的恋爱人格类型。

共有 **27 种普通人格 + 1 种隐藏人格**，每种人格都用一个一看就懂的关键词命名（如 `WIFI`、`GHOST`、`520`、`ICE`、`DRAMA`），配有专属 Emoji 形象、段子化描述和分享文案。

## 维度体系

**5 个行为模型 × 3 个子维度 = 15 个评估维度**，每个维度 L/M/H 三档。

| 模型        | 子维度                                    |
| ----------- | ----------------------------------------- |
| 💗 依恋模式 | A1 安全感需求 · A2 独立程度 · A3 信任基线 |
| 💬 表达风格 | E1 情感表达 · E2 冲突处理 · E3 浪漫指数   |
| 🎵 相处节奏 | P1 亲密频率 · P2 社交边界 · P3 生活融合度 |
| 🔮 未来规划 | F1 承诺态度 · F2 家庭观念 · F3 成长意愿   |
| 💝 爱的语言 | L1 关怀方式 · L2 礼物态度 · L3 陪伴质量   |

## 匹配算法

1. 每道题 1/2/3 分，每维度 2 题，分值范围 2-6
2. 分数转等级：≤3 → L，4 → M，≥5 → H
3. 15 维度等级转为数值向量（L=0, M=1, H=2）
4. 与 27 种人格的标准向量计算**曼哈顿距离**
5. 距离最小者为匹配人格，同时计算匹配度百分比

## 隐藏人格触发

答完 30 道正式题后，会出现 2 道不计分的彩蛋题：

1. 「你现在的感情状态？」→ 选择「一言难尽...」触发第 2 题
2. 「一言难尽是怎么个难尽法？」→ 选择「分分合合第N次了」

两个条件同时满足 → 解锁隐藏人格 **🔄 LOOP（纠缠不清体质）**，匹配度直接 100%。

## 技术栈

- **框架**：Next.js 16 + TypeScript
- **样式**：Tailwind CSS 4
- **部署**：静态导出，可部署至任意平台
- **形象**：Emoji 表情符号（无图片依赖，零资源开销）
- **海报**：Canvas + QRCode 生成分享海报
- 纯前端计算，无后端依赖，无数据收集

│   └── RadarChart.tsx     # 五维雷达图
├── data/
│   ├── questions.ts      # 30 道题目
│   ├── personalities.ts  # 27+1 种人格定义

## 本地开发

```bash
npm install
npm run dev        # 启动开发服务器 http://localhost:3000
npm run build      # 构建（静态导出到 out/）
```

## 部署

项目配置了 `output: "export"`，构建后生成纯静态文件，可部署到任意静态托管平台：

```bash
npm run build
# 将 out/ 目录部署到 Vercel / Netlify / Cloudflare Pages / EdgeOne Pages 等
```
